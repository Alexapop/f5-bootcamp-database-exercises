# SQL - Sales

## Descripción

Ejercicio de SQL para crear consultas sobre un conjunto de datos de ventas en una base de datos MySQL.

## Objetivos

- Crear una base de datos MySQL en Docker
- Crear la tabla `sales` y cargar los datos correspondientes
- Escribir consultas SQL para mostrar información por categoría y por región
- Crear un script para obtener solo las sub categorías que empiezan por la letra "C"
- Crear un script para obtener la cantidad total de unidades vendidas
- Crear un script para obtener la unidades totales del continente americano
- Crea un repositorio con el Readme
- Aloja los scripts en tu repositorio
